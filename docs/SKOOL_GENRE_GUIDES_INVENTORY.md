## Summary Statistics

- **Total genres ready for conversion:** 191
- **Most common file pattern:** Genre Guide (docx) + Story Devices Lexicon (docx) + Style Guides (Gemini + Perplexity in docx)
- **Secondary pattern:** Novelcrafter templates (.md) + Quick Reference Guides (.md) + Skool images (.jpg)
- **Unique formats found:** docx (primary), md, jpg, png, pdf, csv, gif
- **Estimated source files to parse:** 800+ documents across all genres

## Notes for Conversion

- Most genres have 4-7 source documents to synthesize into BookClaw's 7-file schema
- "Comprehensive Guide" documents typically contain reader expectations + beats + must-haves
- "Story Devices Lexicon" maps well to tropes
- "Style Guide" files often include themes and genre-killers
- Novelcrafter templates provide quick reference frames for all sections
- Some genres (like Dark Academia) include supplementary data files (CSV) with market research
- A few genres have multiple variants (e.g., 3 Romantasy versions) that can be consolidated or kept separate
