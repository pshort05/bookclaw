## Obligatory Scenes

- Use this checklist for each scene you write:
-  [ ] Scene purpose clear (advance plot, develop character, build relationship, establish atmosphere)?
-  [ ] Which POV character is primary?
-  [ ] What specific location is this scene set in?
-  [ ] Which secondary character(s) appear, if any?
-  [ ] What's the tonal intention (spooky, cozy, funny, emotional)?
-  [ ] Is this scene investigation, character bonding, or atmospheric setup?
-  [ ] Sensory details specific to this location included?
-  [ ] Character voice distinct in dialogue and internal monologue?
-  [ ] If supernatural element present, is it mystery or revelation?
-  [ ] Is humor balanced against any spooky elements?
-  [ ] Is pacing appropriate to scene purpose?
-  [ ] Does scene advance emotional arc, not just plot?
-  [ ] Are secondary characters developed (personality, voice, depth)?
-  [ ] Does scene maintain genre tone (warm, possibly funny, not traumatizing)?
-  [ ] Would reader experience whimsy and slight unease simultaneously?
-  [ ] Is protagonist agentic (making choices, not passive)?
-  [ ] Does scene end with either mystery deepened or relationship strengthened?
-  [ ] Could this scene be cut without major plot loss? (If yes, consider cutting)
-  [ ] Is sensory grounding present even in eerie moments?
-  [ ] Does dialogue sound natural and reveal character?
