Obligatory scenes and key beats drive this genre forward.

## Obligatory Scenes

- Before Writing Scene**:
-  [ ] What is this scene's military function? (Establishing hierarchy, advancing skill progression, creating moral dilemma, unit bonding, etc.)
-  [ ] What rank/organizational positions are characters in?
-  [ ] What constraints does hierarchy place on character action?
-  [ ] What will character learn or change through this scene?
- During Scene**:
-  [ ] Does dialogue reflect rank relationships naturally?
-  [ ] Are specific sensory details grounding scene in military context?
-  [ ] Is character voice consistent with military training/background?
-  [ ] Does action show character skill level realistically?
-  [ ] Is organizational hierarchy actually constraining or enabling action?
- After Scene**:
-  [ ] Does scene advance character progression meaningfully?
-  [ ] Does scene show cost or consequence (emotional, organizational, or physical)?
-  [ ] Does scene deepen unit bonds or reveal organizational fractures?
-  [ ] Could this scene only work in military SF-Fantasy (or is it generic?)?
-  [ ] Does scene balance action with character interiority?
- 
- 1. **Hierarchy Constraint**: Write scene where character wants to take action but rank/position prevents it. Show creative problem-solving around constraint.
- 2. **Mentorship Moment**: Write training scene showing character failing at skill, mentor's correction, and character's improvement through practice.
- 3. **Moral Dilemma**: Write scene where character receives orders creating genuine internal conflict (no easy answer).
- 4. **Unit Bonding**: Write scene showing why soldiers would die for these specific unit-mates (not abstract duty).
- 5. **Rank Conversation**: Write dialogue between characters of different ranks showing how hierarchy shapes what's said and how it's said.
- 6. **Organizational Friction**: Write scene showing military bureaucracy creating problem (supply issue, communication failure, personality conflict between units).
- 7. **Progression Celebration**: Write moment showing character earning new rank/skill/responsibility and what it costs them personally.
- 
-  [ ] Military hierarchy is functionally present (not just window dressing)
-  [ ] Character voice reflects military training/experience
-  [ ] Dialogue sounds like actual soldiers, not exposition delivery
-  [ ] Sensory details ground action in military context
-  [ ] Scene advances character progression or reveals organizational dynamics
-  [ ] Costs/consequences are visible (not free victories)
-  [ ] Unit relationships matter emotionally
-  [ ] Reader understands character's organizational constraints
