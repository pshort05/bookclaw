*Avoid these patterns—they undermine scientific rigor that defines portal science fiction.*

- ✗ **Magic Portals** — portals function like magic without technological explanation; loses sci-fi grounding
- ✗ **Inconsistent Physics** — portal behaves differently when plot requires; readers versed in physics notice rule-breaking
- ✗ **Consequence-Free Contact** — portal access creates no real problems; worlds coexist without political/economic/military competition
- ✗ **Instant Communication** — language barriers disappear, translation is perfect, misunderstanding never occurs; undermines cultural collision stakes
- ✗ **Easy Portal Control** — single protagonist controls access; ignores geopolitical reality that governments would fight for portal authority
- ✗ **Unexplained World Differences** — alternate world is vaguely described; readers don't understand why/how it's different
- ✗ **Portal as Plot Device** — gateway exists only for action sequences; portal mechanics and implications don't genuinely drive narrative
- ✗ **Humanity Always Superior** — humans/protagonist civilization always dominates alternate worlds; no genuine threat or learning from contact
