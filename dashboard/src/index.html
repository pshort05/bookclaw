Welcome — connect at least one AI provider to get started

BookClaw needs an API key (paid) or a local Ollama install (free) to write. Pick one:

  • Free + cloud: Get a Gemini API key from Google AI Studio (link below)
  • Free + local: Install Ollama and run ollama serve — auto-detected
  • Paid + best quality: Get a Claude or OpenAI key
API Keys (Cloud Providers)

Keys are encrypted with AES-256-GCM in the local vault. Values are never sent anywhere except the provider's own API. Click Get key next to a provider to open its sign-up page.

Need to load keys from a shared folder (VM / Docker)?

OpenRouter Model

If you've added an OpenRouter API key above, pick which model BookClaw should use through it. Examples: anthropic/claude-sonnet-4-5, openai/gpt-4o, meta-llama/llama-3.3-70b-instruct, google/gemini-2.5-flash. Full list at openrouter.ai/models.

Local AI (Ollama) — free, no key

Ollama runs models on your own machine. No API key. Auto-detected when running. Default endpoint: http://localhost:11434
